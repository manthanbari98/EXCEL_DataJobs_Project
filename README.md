# EXCEL_DataJobs_Project
Excel dashboard project using Power Query and Power Pivot, along with Excel add-ins, to transform data, build a data model, and create interactive dashboards.

Welcome to the **EXCEL_DataJobs_Project** repository!

This project demonstrates a comprehensive Excel-based analytics solution, from data preparation and modeling using Power Query and Power Pivot to generating actionable insights through an interactive dashboard.

## Project Requirements

### Building Data Warehouse (Data Engineering)

### Objectives
Develop a data warehouse using SQL Server to consolidate sales data, enabling analytical reporting and informed decision making using python as orchestrator to perform reloads.

### Specifications
- **Data Sources**: Import data provided as xlxs.
- **Data Quality**: Cleanse and resolve data quality issues prior to analysis.
- **Integration**: combining source tables and using excel add-ins(What-if Analysis, Forecast sheet, Solver, Data Analysis) to help forecast trends and creating user friendly data charts.

### Objectives
Develop charts to  deliver insights into:
- **Job Trends**
- **Job Countries**
- **Salary integrations**
